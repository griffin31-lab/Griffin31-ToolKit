# Griffin31 ToolKit

Free tools by Griffin31 to help M365 admins stay ahead of Microsoft changes and strengthen their security posture.

These tools complement the [Griffin31](https://griffin31.com) SaaS platform for M365 security posture management, alerts, audit, and compliance.

## Tools

| Tool | Description |
|------|-------------|
| [CA-Update-AffectedApps](CA-Update-AffectedApps/) | Identifies tenant apps affected by Microsoft's upcoming Conditional Access enforcement change (May 2026). Scans for apps using only basic OIDC/directory scopes that will face new CA policy enforcement. |
