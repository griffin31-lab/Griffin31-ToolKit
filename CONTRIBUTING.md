# Contributing

Thanks for your interest in Griffin31 ToolKit!

## How to contribute

1. **Report bugs** — Open an issue with steps to reproduce
2. **Suggest tools or features** — Use the Discussions tab
3. **Submit fixes** — Fork the repo, make your change, and open a pull request

## Guidelines

- Keep scripts self-contained (one tool per folder)
- Don't commit secrets, tenant IDs, or customer data
- Test your changes before submitting
- Follow the existing folder and naming conventions

## Questions?

Use the [Discussions](https://github.com/griffin31-lab/Griffin31-ToolKit/discussions) tab.
