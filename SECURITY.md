# Security Policy

## Reporting a Vulnerability

If you discover a security issue in any Griffin31 ToolKit script, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email us at: **security@griffin31.com**

We will acknowledge your report within 48 hours and work with you to resolve the issue.

## Scope

This policy applies to all tools in this repository. These tools run locally and connect to Microsoft cloud services using your own credentials — they do not collect, store, or transmit data to Griffin31 or any third party.

## Supported Versions

Only the latest version on the `main` branch is supported.
